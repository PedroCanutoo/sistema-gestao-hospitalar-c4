# Sistema de Gestão Hospitalar - Modelo C4

Projeto acadêmico desenvolvido na disciplina de Engenharia de Requisitos do curso de Sistemas de Informação da PUC Minas.

## Sobre o Projeto

O objetivo deste trabalho foi modelar a arquitetura de um Sistema de Gestão Hospitalar utilizando o Modelo C4, representando diferentes níveis de abstração para facilitar a compreensão da estrutura, dos componentes e das interações do sistema.

A proposta permitiu aplicar conceitos de Engenharia de Requisitos, Arquitetura de Software e Modelagem de Sistemas, auxiliando na identificação de atores, responsabilidades, contêineres, componentes e elementos de código.

## Níveis Desenvolvidos

* Nível 1 – Diagrama de Contexto
* Nível 2 – Diagrama de Containers
* Nível 3 – Diagrama de Componentes
* Nível 4 – Diagrama de Código

## Conceitos Aplicados

* Engenharia de Requisitos
* Arquitetura de Software
* Modelo C4
* UML
* Separação de Responsabilidades
* Repository Pattern
* DTO Pattern
* Injeção de Dependência
